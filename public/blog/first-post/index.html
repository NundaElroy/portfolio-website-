## üöÄ Connecting a ReactJS App Using TanStack Router to Spring Boot

Building modern web applications often requires connecting a **ReactJS frontend** to a **Spring Boot backend**. In this post, I will show you how to use **TanStack Router** to establish seamless communication between your React application and your Spring Boot backend.

### üìå What We'll Cover

- Setting up a Spring Boot Backend API
- Creating a ReactJS App using Vite
- Configuring TanStack Router for routing
- Making API calls to Spring Boot using Axios

---

### üî® Setting Up the Spring Boot Backend

We'll start by creating a simple Spring Boot backend that exposes a REST API.

**Dependency:** Make sure to include `Spring Web` in your `pom.xml`.

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>

Controller Class:

@RestController
@RequestMapping("/api")
public class GreetingController {

    @GetMapping("/greeting")
    public ResponseEntity<String> getGreeting() {
        return ResponseEntity.ok("Hello from Spring Boot!");
    }
}

Run your Spring Boot application and ensure it’s available at: http://localhost:8080/api/greeting.

üåü Creating the React App #

We’ll use Vite for setting up a React project quickly.

npm create vite@latest my-react-app -- --template react
cd my-react-app
npm install
npm install @tanstack/react-router axios

üìå Setting Up TanStack Router #

Create a Router.tsx file in your src directory.

// src/Router.tsx
import { RouterProvider, Router, Route } from '@tanstack/react-router';
import { useState, useEffect } from 'react';
import axios from 'axios';

const Home = () => {
    const [message, setMessage] = useState('');

    useEffect(() => {
        axios.get('http://localhost:8080/api/greeting')
            .then(response => setMessage(response.data))
            .catch(error => console.error('Error fetching data:', error));
    }, []);

    return (
        <div>
            <h1>Home Page</h1>
            <p>{message}</p>
        </div>
    );
};

const routes = [
    { path: '/', element: <Home /> },
];

const router = new Router({ routes });

const App = () => {
    return (
        <RouterProvider router={router} />
    );
};

export default App;

üîë Explanation #

  • Spring Boot Backend: Simple REST API serving a greeting message.
  • TanStack Router: Handling routing within the React app, replacing React Router with a modern alternative.
  • Axios: Fetching data from the backend.
  • Vite: Fast build tool for React apps.

üåê Running the Application #

  1. Start your Spring Boot application: 
    ./mvnw spring-boot:run
  2. Start your React app: 
    npm run dev
  3. Visit your React app at: 
    http://localhost:5173/

You should see the message from your Spring Boot backend displayed on your React app’s homepage! üéâ
